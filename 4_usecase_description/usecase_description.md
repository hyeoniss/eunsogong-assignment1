# usecase description

---
## 2.1 로그인

description: 사용자는 ID와 비밀번호로 로그인할 수 있다. 관리자는 주어진 ID와 비밀번호로 로그인한다.

| Actor Action | System Response |
|---|---|
| 1. NONE | 2. 로그인 화면을 보여준다. |
| 3. 사용자는 ID와 비밀번호를 입력한다. | 4. 입력된 ID와 비밀번호를 표시한다. | 
| 5. 사용자는 로그인 버튼을 선택한다. | 6. 메인 화면을 보여준다. |

---
## 2.2 로그아웃

description: 사용자는 언제든 로그아웃할 수 있다.

| Actor Action | System Response |
|---|---|
| 1. 사용자는 로그아웃 버튼을 선택한다. | 2. 로그아웃 완료 화면을 보여준다. |

---
## 8. 통계 조회

description: 관리자는 설문 응답 통계 정보를 조회할 수 있다. 검색 단위를 최근 1주일, 1개월, 1년 중에서 선택하면 그 기간 동안 전체 설문 수, 종료된 설문 수, 총 응답 수가 출력된다.

| Actor Action | System Response |
|---|---|
| 1. NONE | 2. 통계 조회 화면을 보여준다. |
| 3. 관리자는 조회 기간을 선택한다. | 4. 선택한 조회 기간을 표시한다. |
| 5. 관리자는 조회 버튼을 선택한다. | 6. 전체 설문 수, 종료된 설문 수, 총 응답 수를 보여준다. |
